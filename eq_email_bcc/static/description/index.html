Email Bcc

Key Features:

  • Allow users to send the emails as Bcc to configurable email addresses.
  • Dynamic Configuration for any models and Mail compose Wizard.
  • Mail template wise configuration.

Overview

  • For configure Bcc emails, Go to menu Settings > Technical > Email > Templates. Open form view and go to tab Email Configuration. There is option for set the Bcc email recipients.
  • While sending the Emails, into the open pop-up it show you the Bcc email recipients that is configure inside the mail template.
  • From the Emails menu, user can see the Bcc email recipients And also user can directly set the Bcc email recipients from here and send the emails to bcc email addresses.
